# Style Guide

## Voice

The journal should read like a campaign chronicle with a table-side sense of humor. Aim for clear adventure prose with enough personality to remember why the session was funny, tense, or strange.

Use:

- Vivid but compact descriptions.
- Specific character names and party references.
- Short summaries of rules outcomes when they matter.
- Table jokes, quotes, and odd decisions when they shaped the session.

Avoid:

- Long mechanical play-by-play summaries.
- Overexplaining Frosthaven rules.
- Spoilers that have not appeared in the campaign log yet.
- Rewriting the party into a more generic fantasy group.

## Session Entry Structure

Use this order unless the user asks for something different:

1. Week heading.
2. Scenario number and name.
3. Date played, if known.
4. Image, if one exists.
5. Brief outcome summary.
6. Narrative session recap.
7. Character moments.
8. Rewards, unlocks, retirements, level-ups, or settlement changes.
9. Funny quotes or banter, if available.

## Headings

- Use `#` for page titles.
- Use `##` for major page sections.
- Use `###` for individual story threads, character subsections, or smaller entry sections.
- Logbook week headings should be easy to link to from `index.md`.

Preferred format:

```md
## Week 017 - Scenario Name
```

## Names And Terms

- Preserve exact character names, including punctuation and capitalization.
- Keep Frosthaven terms consistent with the source material and existing files.
- Use party-specific phrasing from the current journal when available.

## Images

- Store logbook images in `assets/images/`.
- Use lowercase, descriptive filenames.
- Prefer this filename pattern:

```text
logbook-week-###-scenario-name.png
```

- Use clear alt text:

```md
![Week 017: Scenario Name](assets/images/logbook-week-017-scenario-name.png)
```

## Markdown

- Prefer standard Markdown tables, lists, headings, and relative links.
- Keep line wrapping readable.
- Do not add decorative HTML unless it is needed for GitHub Pages rendering.
- Use horizontal rules sparingly.
